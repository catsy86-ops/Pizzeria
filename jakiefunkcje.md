# Lista funkcji aplikacji Pizzeria

## Dla klienta

- [ ] Przeglądanie menu i filtrowanie (np. wegetariańskie, ostre)
- [ ] Dodawanie produktów do koszyka i modyfikacja wielkości zamówienia
- [ ] Składanie zamówienia (formularz dostawy) i wybór formy płatności
- [ ] Rejestracja, logowanie oraz podgląd historii zamówień

## Dla obsługi / administratora

- [ ] Panel zarządzania przychodzącymi zamówieniami (zmiana statusów)
- [ ] Edycja menu (dodawanie nowych pizz, zarządzanie cenami i składnikami)
